Usage of menus:

Radial menu:

•	Right click once to popup the menu
•	Left click on an item to select it
•	Right click to reset menu

Marking menu:

•	Hold left mouse button to popup menu
•	Drag pointer to category and release left mouse button to select it
•	Release left mouse button to close menu
•	Advanced users can draw the entire stroke to select item instead of waiting for the menu popup.
If at any time you need help on how to use the menus, click on the help button available at the top of the page for each trial.
